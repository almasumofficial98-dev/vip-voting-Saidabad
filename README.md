# VIP Student Council Voting Website 2026-27

A premium, real-time student council election website built with Vanilla HTML, CSS, and JS. It connects directly to Google Firebase Firestore to securely register votes and display live results on an interactive dashboard.

## Features
- **Live Dashboard**: Displays real-time voting counts, leaderboards, and progress bars.
- **Dynamic Leader Badges**: Highlights the candidate currently leading each category with a crown icon (👑).
- **Branded Design**: Implemented with a custom light-theme matching VIP's International School branding colors.
- **Sleek Custom Dialogs**: Confirmations and success screens run on premium custom modals with Lottie animations.
- **No-Auth Voter Flow**: Simple voting mechanism designed for supervised offline elections.
